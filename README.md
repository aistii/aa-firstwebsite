# appAcademy Open | First Website!
For appAcademy Open's first capstone project (HTML and CSS Capstone Project)!
This involves:
- Building four pages:
  - All with a _navigation bar_;
  - One page acting as an "article";
  - One page acting as a field (no submission function, however; this is to test out knowledge of HTML and CSS);
  - One page of lists that link to other websites;
  - and one page that uses a layout to organize images into a grid
